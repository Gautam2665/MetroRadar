# Design System: Spacing & Layout

Consistent grid layouts and spacing help convey information hierarchy and ensure the platform feels professional and clean.

## Padding & Margins

MetroRadar uses standard tailwind spacing values:
-   `4` (1rem / 16px) — Standard inner cards, spacing between list elements.
-   `6` (1.5rem / 24px) — Dashboard panels padding.
-   `8` (2rem / 32px) — Major layout component margins.

## Layout Containers

-   **Sidebar Width**: Fixed at `380px` (`w-96`) to ensure text remains highly readable.
-   **Header Height**: Fixed at `64px` (`h-16`) for consistent navigation.
-   **Grid Gap**: Standard gap of `4` (`gap-4`) or `6` (`gap-6`) across responsive dashboard slots.
