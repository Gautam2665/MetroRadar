# Design System: Typography

Clean sans-serif fonts allow developers and transit operators to scan large volumes of data without visual fatigue.

## Google Fonts Integration

-   **Primary Interface Font**: **Inter** (loaded via Next.js `next/font/google`).
-   **Fallback Family**: Sans-serif.

## Font Sizes & Weights

-   **Dashboard Titles**: `text-4xl` (`36px`) / `font-extrabold`.
-   **Section Titles**: `text-lg` (`18px`) / `font-bold` / `tracking-wider` / `uppercase`.
-   **Body Text**: `text-sm` (`14px`) / `font-normal` or `font-medium`.
-   **Diagnostic / Diagnostics HUD**: `font-mono` / `text-xs` (`12px`) for align-critical numeric outputs.
