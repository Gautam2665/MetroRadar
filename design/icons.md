# Design System: Icons

We use **Lucide React** to supply lightweight, vector-based diagnostic icons.

## Symbol Meanings

*   `Map`: Map / layer management.
*   `Search`: Transit search bar.
*   `Settings`: Diagnostics dashboard / import console.
*   `Activity`: Diagnostic HUD statistics.
*   `Layers`: Toggling GIS overlays.
*   `Info`: Station overview details.
*   `Train`: Platforms and trip tracks.
*   `Accessibility`: Wheelchair / lift checkups.
*   `Coffee`: Commercial / retail spaces.
*   `Compass`: Navigation and nearby queries.
